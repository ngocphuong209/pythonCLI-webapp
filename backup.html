How to Use

1. Writing Code

Use the code editor to write your Python code. The editor supports syntax highlighting and auto-completion to make coding easier.

  • Type your Python code in the main editor area
  • Use Ctrl + Space for auto-completion suggestions
  • Use Ctrl + / or Cmd + / to comment/uncomment lines

2. Running Your Code

Execute your Python code and see the results instantly.

  • Click the "Run" button or use Ctrl + Enter to execute your code
  • The output will appear in the "Results" section below the editor
  • If your code has errors, they will be displayed in red in the output area

3. Using Advanced Options

Customize your coding experience with advanced features.

  • Click the "Settings" button to access advanced options
  • Enable "Require keyboard input" if your code needs user input
  • Use the input text area to provide multiple input values, one per line
  • Check the list of pre-installed libraries you can use in your code

4. Additional Features

Explore other helpful features of the playground.

  • Use the "Clear" button to reset the editor and output
  • Click "Save" to download your code as a .py file
  • Share your code with others using the "Share" button
  • Toggle between light and dark themes with the theme button
  • Enter fullscreen mode for a distraction-free coding experience

5. Tips for Effective Use

  • Start with small code snippets and gradually build more complex programs
  • Use comments to explain your code and make it easier to understand
  • Experiment with different Python libraries to expand your coding capabilities
  • Save your code regularly to avoid losing your work
  • Use the share feature to get help or collaborate with others
